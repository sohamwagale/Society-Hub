✅ Step 4 — Re-copy Without venv (Clean Way)
🔥 BEST METHOD (Professional Way)

Instead of manually moving folders every time, use this:

From Windows:

scp -i "C:\Users\soham\.ssh\soc.pem" -r `
E:\Programs\projects\soc\backend\* `
ec2-user@13.126.10.73:/home/ec2-user/backend

Notice:

backend\*

That copies only contents, not the entire folder including hidden stuff like venv (if excluded).

If venv is inside backend, it will still copy — so better approach below 👇

💎 BEST PRACTICE METHOD (Recommended)

Create a .gitignore-style exclude using tar:

From Windows:

cd E:\Programs\projects\soc

tar --exclude="backend/venv" -czf backend.tar.gz backend

Then upload:

scp -i "C:\Users\soham\.ssh\soc.pem" backend.tar.gz ec2-user@13.126.10.73:/home/ec2-user/

SSH back in:

tar -xzf backend.tar.gz
rm backend.tar.gz

This guarantees venv is not copied.
